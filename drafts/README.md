# OIA Drafts

This directory holds work-in-progress content that has not yet been reviewed or published.

## Lifecycle

```
drafts/          →  review  →  articles/    (published)
                             →  discard     (deleted)
```

A file stays in `drafts/` until it is either:

- **Promoted** to `articles/` after a review pass (content reviewed, language polished)
- **Discarded** — deleted when the idea does not warrant publication

Draft files are not linked from the site or README. They are working documents only.

## Naming

Follow the same kebab-case, English convention as `articles/`:
`drafts/topic-of-the-draft.md`

Exception: LinkedIn articles written in German retain German titles by convention (see CONVENTIONS.md §2.2).
